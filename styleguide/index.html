Bootstrap extendable

Alpha background

Transparent background with range from 10 to 95 and step equals 5. The class cas hover and focus effect on both <button> and <a> elements.

Html example

<button class="bg-primary-50">Click</button>
<div class="bg-light-85"></div>

Style output example

...
.bg-light-80 {
    background-color: rgba(248, 249, 250, 80) !important;
}
...
​

Width and height breakpoint based

Width class for each breakpoint.

Html example

<div class="w-15"></div>
<div class="w-lg-50"></div>
<div class="h-md-10"></div>

Style output example

...
@media (min-width: 1200px) {
    .w-lg-10 {
        width: 10% !important;
    }
}
...

Transition

Apply transition classes

Available classes

.transition

Html example

<div class="transition"></div>

Style output example

...
.transition {
    transition: all 350ms linear;
}
...
​

Transition speed

Adjust the transition timing class in millisecond

Available classes

.transition-350 // 350ms

Html example

<div class="transition transition-350"></div>

Style output example

...
.transition-350 {
    transition-duration: 350ms;
}
...
​

Transition property

Apply the transition effect by property class

Available classes

.transition-width // apply for transition on width only

Html example

<div class="transition transition-width"></div>

Style output example

...
.transition-width {
    transition-property: width !important;
}
...
