Alpha background Display typography Position Position alignments Transition Transition speed Transition property Width Width and height breakpoint based Height

Bootstrap extendable

Alpha background

Transparent background with range from 10 to 95 and step equals 5. The class cas hover and focus effect on both <button> and <a> elements.

Html example

<button class="bg-primary-50">Click</button>
<div class="bg-light-85"></div>

Style output example

...
.bg-light-80 {
    background-color: rgba(248, 249, 250, 80) !important;
}
...
​

Display typography

Display-1 to 4 with breakpoints.

Pattern: display-*-*

  • First parameter: sm, md, lg, xl
  • Second parameter: 1, 2, 3, 4

Example: display-lg-1

It will display on large displays and greater

Position

Apply position classes with breakpoint supported

Html example

<div class="position-lg-absolute"></div>

Style output example

...
.position-lg-absolute {
    position: absolute;
}
...
​

Transition

Apply transition classes

Available classes

.transition

Html example

<div class="transition"></div>

Style output example

...
.transition {
    transition: all 350ms linear;
}
...
​

Transition speed

Adjust the transition timing class in millisecond

Available classes

.transition-350 // 350ms

Html example

<div class="transition transition-350"></div>

Style output example

...
.transition-350 {
    transition-duration: 350ms;
}
...
​

Transition property

Apply the transition effect by property class

Available classes

.transition-width // apply for transition on width only

Html example

<div class="transition transition-width"></div>

Style output example

...
.transition-width {
    transition-property: width !important;
}
...
​

Width

Width class in pixel for with breakpoint support. Range from 0 to 150 with step 5

Html example

<div class="width-50"></div>
<div class="width-lg-10"></div>

Style output example

...
.width-10 {
    width: 10px !important;
}
...
​

Width and height breakpoint based

Width class for each breakpoint.

Html example

<div class="w-15"></div>
<div class="w-lg-50"></div>
<div class="h-md-10"></div>

Style output example

...
@media (min-width: 1200px) {
    .w-lg-10 {
        width: 10% !important;
    }
}
...
​

Height

Height class in pixel for with breakpoint support. Range from 0 to 150 with step 5

Html example

<div class="height-50"></div>
<div class="height-lg-10"></div>

Style output example

...
.height-10 {
    height: 10px !important;
}
...
​

Height class in vh for with breakpoint support. Range from 50 to 100 with step 5

Html example

<div class="hv-50"></div>
<div class="hv-lg-10"></div>

Style output example

...
.hv-10 {
    hv: 10vh !important;
}
...
